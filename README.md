# 📁 Формат входных данных

Программа работает с JSON-файлами следующей структуры:

```json
{
  "objects": ["кошка", "собака", "рыба"],
  "attributes": ["млекопитающее", "хищник", "плавает", "домашнее"],
  "context": [
    [1, 1, 0, 1],
    [1, 1, 0, 0],
    [0, 1, 1, 0]
  ]
}
```

### Где:

* `objects` — список объектов
* `attributes` — список признаков
* `context` — матрица инцидентности

  * `1` — объект обладает признаком
  * `0` — не обладает

## 📚 Теоретическая справка

### Формальный контекст

Тройка **(G, M, I)**, где:
- **G** — множество объектов
- **M** — множество признаков
- **I ⊆ G × M** — отношение инцидентности

### Формальное понятие

Пара **(A, B)**, где:
- **A ⊆ G** — объем (extent)
- **B ⊆ M** — содержание (intent)

Условия:
- `A' = B` (объекты из A имеют все признаки из B)
- `B' = A` (признаки из B присущи всем объектам из A)

### Алгоритм Van Der Merwe, Kourie (2002)

Рекурсивный алгоритм построения всех формальных понятий:
1. Начинаем с нижнего понятия (все объекты, их общие признаки)
2. Генерируем новые, добавляя признаки
3. Проверяем уникальность
4. Продолжаем рекурсивно

## 🐛 Возможные ошибки и решения

| Ошибка | Причина | Решение |
|--------|---------|---------|
| Файл не найден | Неверный путь | Проверьте путь |
| Отсутствуют обязательные ключи | Неверный JSON | Проверьте структуру |
| Сначала загрузите контекст | Контекст не загружен | Выполните `load_context` |
| Нет результатов для экспорта | Не выполнено построение | Выполните `build_set` или `build_bitset` |
| Значение 2 в матрице | Некорректные данные | Используйте 0 и 1 |
| Список объектов пуст | Пустой JSON | Добавьте объекты |
| Объект 'X' не найден | Несуществующий объект | Проверьте список |

### Результаты фаззинга

| № | Тест | Итераций | Время | Результат | Найденная ошибка |
|---|------|----------|-------|-----------|------------------|
| 1 | Устойчивость `build_concepts_set` | 1000 | 2.35с | ✅ | — |
| 2 | Свойство замыкания | 1000 | 2.41с | ❌ | StopIteration при пустом множестве |
| 3 | Устойчивость `build_concepts_bitset` | 1000 | 1.89с | ✅ | — |
| 4 | Согласованность Set/BitSet | 500 | 3.12с | ✅ | — |
| 5 | Обработка пустого множества | 500 | 1.56с | ❌ | StopIteration |

**Найденная ошибка:**
```python
# fca/context.py, метод closure
def closure(self, objects: Set[str]) -> Set[str]:
    # ОШИБКА: проверка пустого множества удалена
    # if not objects:
    #     return set(self.attributes)
    
    first = next(iter(objects))  # StopIteration при objects = set()
    # ...
```

**Воспроизводящий пример:**
```python
context = FormalContext(objects=['o0'], attributes=['a0'], matrix=[[0]])
context.closure(set())  # StopIteration
```
